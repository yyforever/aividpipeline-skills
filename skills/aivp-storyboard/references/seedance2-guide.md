# Seedance 2.0 分镜专用指南

基于字节官方使用手册 + elementsix/seedance-storyboard skill + 社区实战总结（2026.02）。

## 核心能力参数

| 输入类型 | 格式 | 数量限制 | 大小限制 |
|---------|------|---------|---------|
| 图片 | jpeg/png/webp/bmp/tiff/gif | ≤ 9 张 | < 30 MB |
| 视频 | mp4/mov | ≤ 3 个，总时长 2-15s | < 50 MB |
| 音频 | mp3/wav | ≤ 3 个，总时长 ≤ 15s | < 15 MB |
| 文本 | 自然语言 | — | 30-200 词最佳 |

**混合输入总上限：12 个文件**

两种功能模式：
- **全能参考（omni_reference）**：多模态混合输入，@ 语法分配素材角色
- **首尾帧（first_last_frames）**：首帧/首末帧→视频，适合精确控制

## 分镜提示词结构（Seedance 专用）

### 标准格式

```
【整体描述】
{风格} + {时长}秒 + {画面比例} + {整体氛围/色调}

【分镜描述】
0-3秒：{景别}{运镜}，{画面内容}，{主体动作}，{光影效果}
3-7秒：{景别}{运镜}，{画面内容}，{主体动作}，{光影效果}
7-11秒：...
11-15秒：...

【声音说明】
{配乐风格} + {音效} + {对白/旁白}

【参考素材说明】（如有）
@图片1 作为角色形象参考
@图片2 作为场景风格参考
@视频1 参考运镜和动作
@音频1 作为背景音乐
```

### 单镜头提示词公式

```
Subject + Action + Camera + Style + Constraints
```

详细卡片格式：
```
Subject: [人物/物体, 年龄/材质]
Action: [具体动词短语, 现在时]
Camera: [景别] + [运镜] + [角度], [焦段感觉]
Style: [一个视觉锚点: 电影/工艺/导演风格], [光线], [色彩处理]
Constraints: [禁止项], [帧率/节奏], [时长], [一致性要求]
```

## 运镜关键词速查

| 运镜类型 | 关键词 | 效果 |
|---------|--------|------|
| 推近 | dolly-in, 推镜, push in | 建立张力/亲密感 |
| 拉远 | dolly-out, 拉镜, pull out | 揭示环境 |
| 横移 | track left/right, 移镜 | 伴随主体横向移动 |
| 水平摇 | pan left/right, 摇镜 | 原地水平旋转 |
| 升降 | crane up/down, 升降镜头 | 垂直移动 |
| 环绕 | orbit, 环绕镜头, 360 orbit | 围绕主体旋转 |
| 手持 | handheld, 手持 | 微晃，纪实感 |
| 稳定器 | gimbal, 稳定器 | 丝滑平稳 |
| 变焦 | dolly zoom, 希区柯克变焦 | 经典眩晕效果 |
| 快摇 | whip pan | 急速水平扫过 |
| 航拍 | aerial sweep, drone shot | 高空俯瞰全景 |

**速度修饰**：slow / medium / fast + 距离描述（如 "slow dolly-in, 1-2 feet"）

**复合运镜**：写成时间节拍，不要塞进一个子句
```
❌ "推镜同时向右摇"
✅ "0-2秒缓慢推镜，2-4秒轻柔向右摇摄"
```

## 景别 × 运镜匹配建议

| 景别 | 适用场景 | 推荐运镜 | 注意 |
|------|---------|---------|------|
| 远景/全景 | 建立场景，环境展示 | 慢推 / 固定 | 避免快摇（产生拖影） |
| 中景 | 对话，UGC | 手持=个人感，稳定器=精致感 | 最安全的景别 |
| 近景/特写 | 细节，情绪 | 微推 | 摇摄在近景很突兀；长焦虚化背景 |

## 八大铁律

### 1. 一镜一动词
每个镜头段（时间片段）只用一个核心动作动词。多动词让模型混乱。

```
❌ "角色奔跑、跳跃、翻滚"
✅ "0-3秒角色全速奔跑，3-5秒跃起翻越栏杆"
```

### 2. 程度副词必加
Seedance 对动作力度非常敏感。

```
❌ "移动" "推" "转"
✅ "缓缓移动" "轻轻推开" "猛然转身"
```

优先使用：慢/缓/轻柔/连续/自然/流畅/丝滑

### 3. 角色不超过 2 人
超过 2 人会导致身份混淆和一致性丢失。

### 4. 提示词长度 30-200 词
太短缺信息，太长模型会忽略细节。

### 5. 无负面提示词
Seedance 不支持负面提示。用正面约束替代：

```
❌ "no blur, no distortion"
✅ "保持面部和服装一致性，无变形，高细节"
```

### 6. 质量后缀必加
每个提示词末尾焊死：
```
4K, 超高清, 细节丰富, 清晰锐利, 电影质感, 自然色彩, 画面稳定
```

### 7. 角色约束词必加
涉及人物时：
```
角色面部稳定不变形，正常人体结构，动作自然流畅
```

### 8. 先短后长
先生成 5-10 秒确认效果，再延长到目标时长。

## @ 素材引用语法

上传文件后模型自动分配标签。在提示词中显式指定用途：

| 用途 | 写法 |
|------|------|
| 设为首帧 | @图片1 作为首帧 |
| 参考运镜 | 参考@视频1 的运镜和转场 |
| 角色形象 | @图片1 中的女子作为主角 |
| 参考动作 | 参考@视频1 的打斗动作 |
| 背景音乐 | @音频1 作为背景音乐 |
| 延长视频 | 将@视频1 延长X秒 |
| 替换角色 | 将@视频1 中的女生换成@图片1 的形象 |
| 完整复刻 | 完全参考@视频1 的所有运镜效果和主角面部表情 |

### 多模态组合策略

**角色一致性控制**：
```
@图片1 作为角色形象参考
@图片2 作为场景风格参考
@图片3 作为首帧画面
```

**运镜动作复刻**：
```
参考@视频1 的希区柯克变焦
参考@视频2 的环绕运镜
主角动作参考@视频3
```

## 从分镜到 Seedance 提示词的转换

aivp-storyboard 的 shot spec 需要转换为 Seedance 格式。对照表：

| Shot Spec 字段 | Seedance 提示词位置 |
|---------------|-------------------|
| ff_desc + lf_desc | → 首尾帧模式的参考图生成依据 |
| motion_desc | → 分镜描述中的动作+运镜部分 |
| shot_type + angle | → 景别+角度关键词 |
| camera movement | → 运镜关键词（用中文/英文均可） |
| duration | → 时间轴标注（0-Xs） |
| variation type | → 决定用首帧模式(small) 还是首尾帧模式(medium/large) |
| dialogue | → 对白文本（Seedance 支持 8 语言原生口型同步） |
| bgm + sfx | → 声音说明段 |
| character refs | → @图片N 角色形象参考 |
| scene bg | → @图片N 场景风格参考 |

### 转换示例

**Shot Spec**:
```
Shot 1.2 | MS | eye-level | slow dolly-in | 5s | var:small | ↑↑
Characters (ff): [@Elena, facing:camera]
First frame: "Medium shot at eye level. Woman (early 30s, auburn hair, gray blazer)
  stands at screen-center, holding a crumpled letter..."
Motion: "Static camera slowly pushing in. The auburn-haired woman's hand
  slowly unclenches, letter begins to slip..."
Dialogue: [Elena, barely audible]: "It was all a lie."
BGM: tense strings
SFX: paper rustling
```

**Seedance 2.0 提示词**:
```
【整体描述】
电影写实风格，5秒，16:9横屏，紧张压抑的室内氛围

【分镜描述】
0-2秒：中景平视缓慢推近，灰色西装外套的短发女人站在画面中央，
  双手紧握一封皱巴巴的信，手指关节发白，暖色侧光从右侧窗户照入
2-5秒：继续缓慢推近至中近景，女人的手缓缓松开，信纸开始从颤抖的
  指间滑落，女人轻声说"这一切都是假的"，表情从紧绷转为崩溃

【声音说明】
紧张的弦乐持续铺底，纸张摩擦声，女人轻声台词

【参考素材说明】
@图片1 作为角色形象参考（确保面部和服装一致性）

角色面部稳定不变形，动作自然流畅，4K超高清，电影质感，画面稳定
```

## 常见问题排查

| 问题 | 解决方案 |
|------|---------|
| 构图对但动作错 | 只改 Camera（景别+一个运镜），保持 Subject/Action 不变 |
| 运动感不对（太晃/太快） | 把 handheld 换成 gimbal（或反之），设显式速度 |
| 风格/色彩漂移 | 用单一更强的视觉锚点替换多个形容词 |
| 手/标签/光晕伪影反复出现 | 换约束词，或改景别/简化主体 |
| 角色变脸 | 加"保持面部和服装一致性，无变形，高细节" |
| 多人互动崩坏 | 限制到 1-2 人，或拆分为多镜头分别生成 |
